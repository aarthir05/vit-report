Module 1: Single Variable Calculus
Rolle's Theorem and Mean Value Theorem
Key Theorems
  • Rolle's Theorem: If \(f\) is continuous on \([a,b]\), differentiable on \((a,b)\), and \(f(a)=f(b)\), then \(\exists\, c \in (a,b)\) such that \(f'(c)=0\)
  • Mean Value Theorem (Lagrange): \(\displaystyle f'(c) = \frac{f(b)-f(a)}{b-a}\) for some \(c \in (a,b)\)
Maxima and Minima (Single Variable)
First and Second Derivative Tests
  • Critical point: \(f'(x)=0\) or \(f'(x)\) undefined
  • Second derivative test: If \(f'(c)=0\) and \(f''(c)>0\) → local minimum; \(f''(c)<0\) → local maximum
  • Concave up: \(f''(x)>0\); Concave down: \(f''(x)<0\)
  • Inflection point: where \(f''(x)=0\) and concavity changes
✏ Solved Example — Maxima/Minima
Find the maxima and minima of \(f(x) = x^3 - 3x^2 - 9x + 5\)
1
Find critical points: \(f'(x) = 3x^2 - 6x - 9 = 3(x^2-2x-3) = 3(x-3)(x+1) = 0\), so \(x = 3, -1\)
2
\(f''(x) = 6x - 6\)
3
At \(x=-1\): \(f''(-1) = -12 < 0\) → local maximum. \(f(-1) = -1-3+9+5 = 10\)
4
At \(x=3\): \(f''(3) = 12 > 0\) → local minimum. \(f(3) = 27-27-27+5 = -22\)
Local max = 10 at x = −1; Local min = −22 at x = 3
Integration — Area and Volumes
Key Formulas
  • Area between curves: \(\displaystyle A = \int_a^b [f(x)-g(x)]\,dx\) where \(f(x) \geq g(x)\)
  • Volume (disc method): \(\displaystyle V = \pi\int_a^b [f(x)]^2\,dx\)
  • Volume (shell method): \(\displaystyle V = 2\pi\int_a^b x\,f(x)\,dx\)

Module 2–3: Multivariable Calculus
Partial Derivatives and Total Differential
Key Definitions
  • Partial derivative \(\partial f/\partial x\): Differentiate w.r.t. \(x\), treating \(y\) as constant
  • Total differential: \(df = \dfrac{\partial f}{\partial x}dx + \dfrac{\partial f}{\partial y}dy\)
  • Jacobian: \(\displaystyle J\!\left(\frac{u,v}{x,y}\right) = \begin{vmatrix} \partial u/\partial x & \partial u/\partial y \\ \partial v/\partial x & \partial v/\partial y \end{vmatrix}\)
  • Chain rule: \(\displaystyle \frac{du}{dt} = \frac{\partial u}{\partial x}\frac{dx}{dt}+\frac{\partial u}{\partial y}\frac{dy}{dt}\)
Taylor's Series for Two Variables
Formula
\[\displaystyle f(a+h,b+k) = f(a,b) + \left(h\frac{\partial}{\partial x}+k\frac{\partial}{\partial y}\right)f + \frac{1}{2!}\left(h\frac{\partial}{\partial x}+k\frac{\partial}{\partial y}\right)^2 f + \cdots\]
Maxima and Minima of f(x,y)
Conditions (Second Order Test)
  • Find critical point: \(f_x = 0\) and \(f_y = 0\)
  • Compute discriminant: \(D = f_{xx}f_{yy} - (f_{xy})^2\)
  • If \(D > 0\) and \(f_{xx} > 0\) → minimum
  • If \(D > 0\) and \(f_{xx} < 0\) → maximum
  • If \(D < 0\) → saddle point
  • If \(D = 0\) → test fails
Lagrange's Multiplier Method

To find extremum of \(f(x,y,z)\) subject to constraint \(g(x,y,z)=0\):

Method
  • Form: \(\nabla f = \lambda\,\nabla g\), i.e. \(f_x = \lambda g_x\), \(f_y = \lambda g_y\), \(f_z = \lambda g_z\)
  • Solve these equations simultaneously with \(g=0\) to find the constrained extremum
✏ Solved Example — Lagrange Multipliers
Find the maximum and minimum of \(f=x^2+y^2\) subject to \(x+y=1\)
1
Set up: \(\nabla f = \lambda\nabla g\) → \(2x = \lambda,\; 2y = \lambda\) → \(x = y\)
2
Substitute into constraint: \(x+x=1\) → \(x=y=\frac{1}{2}\)
Minimum value = \(\frac{1}{4}+\frac{1}{4} = \frac{1}{2}\) at \((1/2, 1/2)\)
Module 4: Multiple Integrals
Key Formulas
  • Double integral (Cartesian): \(\displaystyle \iint_R f(x,y)\,dA = \int_a^b\int_{g_1(x)}^{g_2(x)} f(x,y)\,dy\,dx\)
  • Change to polar: \(x=r\cos\theta,\; y=r\sin\theta,\; dA = r\,dr\,d\theta\)
  • Change of order: Sketch the region, re-identify the limits
  • Triple integral: \(\displaystyle \iiint_V f(x,y,z)\,dV\)
  • Cylindrical coords: \(x=r\cos\theta,\; y=r\sin\theta,\; z=z,\; dV=r\,dr\,d\theta\,dz\)
  • Spherical coords: \(x=\rho\sin\phi\cos\theta,\; y=\rho\sin\phi\sin\theta,\; z=\rho\cos\phi,\; dV=\rho^2\sin\phi\,d\rho\,d\phi\,d\theta\)
Module 5: Special Functions
Beta and Gamma Functions
  • \(\Gamma(n) = \displaystyle\int_0^\infty e^{-t}t^{n-1}\,dt\), \(\quad \Gamma(n+1) = n\,\Gamma(n)\), \(\quad \Gamma(1/2) = \sqrt{\pi}\)
  • \(B(m,n) = \displaystyle\int_0^1 x^{m-1}(1-x)^{n-1}\,dx = \frac{\Gamma(m)\Gamma(n)}{\Gamma(m+n)}\)
  • Useful reduction: \(\displaystyle\int_0^{\pi/2}\sin^m\theta\cos^n\theta\,d\theta = \frac{\Gamma\!\left(\frac{m+1}{2}\right)\Gamma\!\left(\frac{n+1}{2}\right)}{2\,\Gamma\!\left(\frac{m+n+2}{2}\right)}\)
  • Error function: \(\displaystyle\text{erf}(x) = \frac{2}{\sqrt{\pi}}\int_0^x e^{-t^2}\,dt\), \(\quad \text{erfc}(x) = 1-\text{erf}(x)\)

Modules 6–7: Vector Calculus
Vector Differentiation
Gradient, Divergence, Curl
  • Gradient: \(\nabla f = \dfrac{\partial f}{\partial x}\hat{i}+\dfrac{\partial f}{\partial y}\hat{j}+\dfrac{\partial f}{\partial z}\hat{k}\)  → points in direction of maximum increase
  • Directional derivative: \(D_{\hat{u}}f = \nabla f \cdot \hat{u}\)
  • Divergence: \(\nabla\cdot\mathbf{F} = \dfrac{\partial F_1}{\partial x}+\dfrac{\partial F_2}{\partial y}+\dfrac{\partial F_3}{\partial z}\)
  • Curl: \(\nabla\times\mathbf{F} = \begin{vmatrix}\hat{i}&\hat{j}&\hat{k}\\ \partial_x&\partial_y&\partial_z\\ F_1&F_2&F_3\end{vmatrix}\)
  • Irrotational: \(\nabla\times\mathbf{F}=\mathbf{0}\)   Solenoidal: \(\nabla\cdot\mathbf{F}=0\)
Vector Integration Theorems
The Three Major Theorems
  • Green's Theorem (2D): \(\displaystyle\oint_C (P\,dx+Q\,dy) = \iint_R\!\left(\frac{\partial Q}{\partial x}-\frac{\partial P}{\partial y}\right)dA\)
  • Stokes' Theorem: \(\displaystyle\oint_C \mathbf{F}\cdot d\mathbf{r} = \iint_S (\nabla\times\mathbf{F})\cdot d\mathbf{S}\)
  • Gauss Divergence Theorem: \(\displaystyle\oiint_S \mathbf{F}\cdot d\mathbf{S} = \iiint_V \nabla\cdot\mathbf{F}\,dV\)
Exam tip — Green's TheoremOften used to evaluate line integrals by converting to double integrals (or vice versa). Always check if the curve is closed and oriented counter-clockwise.
✏ Solved Example — Gauss Divergence Theorem
Evaluate \(\displaystyle\oiint_S \mathbf{F}\cdot\hat{n}\,dS\) where \(\mathbf{F}=x^2\hat{i}+y^2\hat{j}+z^2\hat{k}\) over the unit cube
1
Find divergence: \(\nabla\cdot\mathbf{F} = 2x+2y+2z\)
2
Apply Gauss: \(\displaystyle\iiint_V(2x+2y+2z)\,dV = \int_0^1\!\int_0^1\!\int_0^1(2x+2y+2z)\,dx\,dy\,dz\)
3
By symmetry each integral gives the same value: \(= 3\times\int_0^1\!\int_0^1\!\int_0^1 2x\,dx\,dy\,dz = 3\times 1 = 3\)
Answer = 3

Module 1: Ordinary Differential Equations
2nd Order ODE with Constant Coefficients
General Solution: ay'' + by' + cy = f(x)
  • Solve auxiliary equation \(am^2+bm+c=0\) for roots \(m_1, m_2\)
  • If \(m_1 \neq m_2\) (real): \(y_c = C_1e^{m_1x}+C_2e^{m_2x}\)
  • If \(m_1 = m_2 = m\) (repeated): \(y_c = (C_1+C_2x)e^{mx}\)
  • If \(m = \alpha \pm i\beta\) (complex): \(y_c = e^{\alpha x}(C_1\cos\beta x + C_2\sin\beta x)\)
Particular Integral — Method of Undetermined Coefficients
f(x)Trial Particular Integral
\(e^{ax}\)\(Ae^{ax}\) (if \(a\) is not a root)
\(\sin ax\) or \(\cos ax\)\(A\sin ax + B\cos ax\)
\(x^n\)\(A_nx^n + A_{n-1}x^{n-1}+\cdots+A_0\)
\(e^{ax}\sin bx\)\(e^{ax}(A\sin bx+B\cos bx)\)
Variation of Parameters Formula
\[y_p = y_1\int\frac{-y_2\,f(x)}{W}\,dx + y_2\int\frac{y_1\,f(x)}{W}\,dx\]
where \(W = y_1y_2'-y_2y_1'\) is the Wronskian
✏ Solved — 2nd Order ODE
Solve \(y'' - 3y' + 2y = e^{3x}\)
1
AE: \(m^2-3m+2=0\) → \((m-1)(m-2)=0\) → \(m=1,2\). So \(y_c=C_1e^x+C_2e^{2x}\)
2
PI: Try \(y_p=Ae^{3x}\). Then \(9Ae^{3x}-9Ae^{3x}+2Ae^{3x}=e^{3x}\) → \(2A=1\) → \(A=\frac{1}{2}\)
\(y = C_1e^x + C_2e^{2x} + \frac{1}{2}e^{3x}\)
Module 2: Partial Differential Equations
Formation and Standard Types
  • Lagrange's PDE: \(Pp + Qq = R\) where \(p=\partial z/\partial x\), \(q=\partial z/\partial y\). Solve using auxiliary equations \(\frac{dx}{P}=\frac{dy}{Q}=\frac{dz}{R}\)
  • Type 1: \(f(p,q)=0\) — assume \(z=ax+by+c\) where \(f(a,b)=0\)
  • Type 2: \(f(z,p,q)=0\) — assume \(z=f(x+ay)\)
  • Separation of variables: Assume \(u(x,t)=X(x)\cdot T(t)\), separate, solve two ODEs

Module 3–4: Laplace Transform
Definition and Properties
  • Definition: \(\mathcal{L}\{f(t)\} = F(s) = \displaystyle\int_0^\infty e^{-st}f(t)\,dt\)
  • Linearity: \(\mathcal{L}\{af+bg\} = aF(s)+bG(s)\)
  • First shift: \(\mathcal{L}\{e^{at}f(t)\} = F(s-a)\)
  • Differentiation: \(\mathcal{L}\{f'(t)\} = sF(s)-f(0)\); \(\mathcal{L}\{f''(t)\} = s^2F(s)-sf(0)-f'(0)\)
  • Integration: \(\mathcal{L}\left\{\int_0^t f(\tau)\,d\tau\right\} = \dfrac{F(s)}{s}\)
  • Convolution: \(\mathcal{L}\{f*g\} = F(s)\cdot G(s)\)
Standard Laplace Transform Table

\(1\)\(1/s\)
\(t^n\)\(n!/s^{n+1}\)
\(e^{at}\)\(1/(s-a)\)
\(\sin at\)\(a/(s^2+a^2)\)
\(\cos at\)\(s/(s^2+a^2)\)
\(\sinh at\)\(a/(s^2-a^2)\)
\(\cosh at\)\(s/(s^2-a^2)\)
\(t\sin at\)\(2as/(s^2+a^2)^2\)
\(u(t-a)\)\(e^{-as}/s\) (Heaviside)

✏ Solved — Solve ODE using Laplace
Solve \(y'' + 4y = \sin 2t\), given \(y(0)=0, y'(0)=0\)
1
Take Laplace: \([s^2Y(s)-0-0]+4Y(s) = \dfrac{2}{s^2+4}\)
2
\(Y(s)(s^2+4) = \dfrac{2}{s^2+4}\) → \(Y(s) = \dfrac{2}{(s^2+4)^2}\)
3
Inverse Laplace: \(\mathcal{L}^{-1}\!\left\{\dfrac{2}{(s^2+4)^2}\right\} = \dfrac{1}{4}(\sin 2t - 2t\cos 2t)\)
\(y = \dfrac{1}{4}(\sin 2t - 2t\cos 2t)\)
Module 5: Fourier Series
Euler's Formulae — Full Range \((-l, l)\)
\[f(x) = \frac{a_0}{2} + \sum_{n=1}^\infty\left(a_n\cos\frac{n\pi x}{l}+b_n\sin\frac{n\pi x}{l}\right)\]
  • \(\displaystyle a_0 = \frac{1}{l}\int_{-l}^l f(x)\,dx\)
  • \(\displaystyle a_n = \frac{1}{l}\int_{-l}^l f(x)\cos\frac{n\pi x}{l}\,dx\)
  • \(\displaystyle b_n = \frac{1}{l}\int_{-l}^l f(x)\sin\frac{n\pi x}{l}\,dx\)
Half-Range Series and Parseval's Identity
  • Half-range cosine: \(a_n = \dfrac{2}{l}\displaystyle\int_0^l f(x)\cos\dfrac{n\pi x}{l}\,dx\), \(b_n=0\)
  • Half-range sine: \(b_n = \dfrac{2}{l}\displaystyle\int_0^l f(x)\sin\dfrac{n\pi x}{l}\,dx\), \(a_0=a_n=0\)
  • Parseval's Identity: \(\dfrac{1}{l}\displaystyle\int_{-l}^l [f(x)]^2\,dx = \dfrac{a_0^2}{2}+\sum_{n=1}^\infty(a_n^2+b_n^2)\)

Module 6: Fourier Transform
Definitions
  • Complex Fourier transform: \(\hat{f}(\omega) = \displaystyle\int_{-\infty}^\infty f(x)e^{-i\omega x}\,dx\)
  • Inverse: \(f(x) = \dfrac{1}{2\pi}\displaystyle\int_{-\infty}^\infty \hat{f}(\omega)e^{i\omega x}\,d\omega\)
  • Fourier sine transform: \(F_s(\omega) = \displaystyle\int_0^\infty f(x)\sin\omega x\,dx\)
  • Fourier cosine transform: \(F_c(\omega) = \displaystyle\int_0^\infty f(x)\cos\omega x\,dx\)
  • Parseval's: \(\displaystyle\int_{-\infty}^\infty |f(x)|^2\,dx = \frac{1}{2\pi}\int_{-\infty}^\infty|\hat{f}(\omega)|^2\,d\omega\)
Module 7: Z-Transform
Definition and Standard Pairs
  • Definition: \(Z\{x(n)\} = X(z) = \displaystyle\sum_{n=0}^\infty x(n)z^{-n}\)
  • \(Z\{1\} = \dfrac{z}{z-1}\)
  • \(Z\{n\} = \dfrac{z}{(z-1)^2}\)
  • \(Z\{a^n\} = \dfrac{z}{z-a}\)
  • \(Z\{\cos n\theta\} = \dfrac{z(z-\cos\theta)}{z^2-2z\cos\theta+1}\)
  • \(Z\{\sin n\theta\} = \dfrac{z\sin\theta}{z^2-2z\cos\theta+1}\)
Properties and Inverse Z-Transform
  • Linearity: \(Z\{ax(n)+by(n)\} = aX(z)+bY(z)\)
  • Time shift: \(Z\{x(n-k)\} = z^{-k}X(z)\)
  • Convolution: \(Z\{x*y\} = X(z)\cdot Y(z)\)
  • Inverse (partial fractions): Expand \(X(z)/z\) in partial fractions, then multiply by \(z\) and use table
✏ Solved — Z-Transform of difference equation
Solve \(y(n+2)-5y(n+1)+6y(n)=0\), \(y(0)=0\), \(y(1)=1\)
1
Take Z-transform: \((z^2-5z+6)Y(z) = z^2y(0)+z[y(1)-5y(0)] = z\)
2
\(Y(z) = \dfrac{z}{(z-2)(z-3)}\) → partial fractions: \(\dfrac{-z}{z-2}+\dfrac{z}{z-3}\)
3
Inverse Z: \(Z^{-1}\!\left\{\dfrac{z}{z-a}\right\} = a^n\)
\(y(n) = 3^n - 2^n\)

Module 1: Analytic Functions
Cauchy-Riemann Equations
  • \(f(z) = u(x,y)+iv(x,y)\) is analytic if and only if:
  • \(\dfrac{\partial u}{\partial x} = \dfrac{\partial v}{\partial y}\) and \(\dfrac{\partial u}{\partial y} = -\dfrac{\partial v}{\partial x}\)
  • Additionally, \(u\) and \(v\) must have continuous first-order partial derivatives
  • If analytic: \(f'(z) = \dfrac{\partial u}{\partial x}+i\dfrac{\partial v}{\partial x} = \dfrac{\partial u}{\partial x}-i\dfrac{\partial u}{\partial y}\)
Harmonic Functions and Conjugate
  • \(u\) is harmonic if \(\nabla^2 u = u_{xx}+u_{yy} = 0\)
  • If \(u\) is harmonic, find \(v\) (harmonic conjugate) using Milne-Thomson method
  • Milne-Thomson: \(f'(z) = \phi_1(z,0) - i\phi_2(z,0)\) where \(\phi_1 = u_x, \phi_2 = u_y\)
✏ Solved — C-R Equations check
Is \(f(z)=z^2\) analytic? Find \(u,v\).
1
\(f(z)=(x+iy)^2 = x^2-y^2 + 2ixy\), so \(u=x^2-y^2\), \(v=2xy\)
2
\(u_x=2x=v_y=2x\) ✓ and \(u_y=-2y = -v_x=-2y\) ✓
C-R equations satisfied everywhere → \(f(z)=z^2\) is analytic everywhere (entire)
Module 2: Conformal Mapping
Elementary Transformations
  • Translation: \(w = z + c\) — shifts the plane
  • Rotation/Magnification: \(w = az\) — rotates by \(\arg(a)\) and scales by \(|a|\)
  • Inversion: \(w = 1/z\) — maps circles to circles/lines
  • Square map: \(w = z^2\) — doubles angles at origin
  • Exponential: \(w = e^z\) — maps horizontal strips to sectors
Bilinear (Möbius) Transformation
\[w = \frac{az+b}{cz+d}, \quad ad-bc \neq 0\]
  • Maps circles and lines to circles and lines
  • Cross-ratio preserved: \((w_1,w_2,w_3,w_4) = (z_1,z_2,z_3,z_4)\)
  • Find transformation by mapping 3 points: use \(\dfrac{(w-w_1)(w_2-w_3)}{(w-w_3)(w_2-w_1)} = \dfrac{(z-z_1)(z_2-z_3)}{(z-z_3)(z_2-z_1)}\)

Module 3: Complex Integration
Taylor and Laurent Series
  • Taylor series (analytic at \(a\)): \(f(z) = \displaystyle\sum_{n=0}^\infty \frac{f^{(n)}(a)}{n!}(z-a)^n\)
  • Laurent series: \(f(z) = \displaystyle\sum_{n=0}^\infty a_n(z-a)^n + \sum_{n=1}^\infty \frac{b_n}{(z-a)^n}\)
  • Residue = coefficient \(b_1\) of \((z-a)^{-1}\) in Laurent series
Singularities and Residues
  • Removable singularity: Laurent series has no negative powers
  • Pole of order \(m\): Laurent series has terms up to \((z-a)^{-m}\)
  • Essential singularity: Infinitely many negative power terms
  • Residue at simple pole: \(\text{Res}(f,a) = \displaystyle\lim_{z\to a}(z-a)f(z)\)
  • Residue at pole of order \(m\): \(\displaystyle\frac{1}{(m-1)!}\lim_{z\to a}\frac{d^{m-1}}{dz^{m-1}}[(z-a)^m f(z)]\)
Cauchy's Theorems
  • Cauchy-Goursat: \(\displaystyle\oint_C f(z)\,dz = 0\) if \(f\) is analytic inside and on \(C\)
  • Cauchy's Integral Formula: \(\displaystyle f^{(n)}(a) = \frac{n!}{2\pi i}\oint_C \frac{f(z)}{(z-a)^{n+1}}\,dz\)
  • Residue Theorem: \(\displaystyle\oint_C f(z)\,dz = 2\pi i\sum_k \text{Res}(f,a_k)\)
✏ Solved — Residue Theorem
Evaluate \(\displaystyle\oint_{|z|=2}\frac{e^z}{z(z-1)}\,dz\)
1
Poles inside \(|z|=2\): \(z=0\) and \(z=1\) (both simple poles)
2
\(\text{Res}(f,0) = \displaystyle\lim_{z\to 0}z\cdot\frac{e^z}{z(z-1)} = \frac{e^0}{0-1} = -1\)
3
\(\text{Res}(f,1) = \displaystyle\lim_{z\to 1}(z-1)\cdot\frac{e^z}{z(z-1)} = \frac{e^1}{1} = e\)
4
Result: \(2\pi i(-1+e) = 2\pi i(e-1)\)
\(\displaystyle\oint = 2\pi i(e-1)\)
Modules 4–5: Vector Spaces and Linear Transformations
Vector Space Definitions
  • Subspace test: Closed under addition and scalar multiplication; contains zero vector
  • Linear independence: \(c_1v_1+c_2v_2+\cdots=0\) implies all \(c_i=0\)
  • Basis: Linearly independent spanning set
  • Dimension: Number of vectors in any basis
  • Rank-Nullity Theorem: \(\text{rank}(A)+\text{nullity}(A) = n\) (number of columns)
Linear Transformations
  • \(T: V\to W\) is linear if \(T(u+v)=T(u)+T(v)\) and \(T(cu)=cT(u)\)
  • Kernel: \(\ker(T) = \{v : T(v)=0\}\)
  • Image: \(\text{Im}(T) = \{T(v) : v\in V\}\)
  • Rank: \(\dim(\text{Im}(T))\); Nullity: \(\dim(\ker T)\)

Module 6: Inner Product Spaces
Gram-Schmidt Orthogonalization
  • Given basis \(\{v_1,v_2,\ldots,v_n\}\), construct orthonormal basis \(\{e_1,e_2,\ldots,e_n\}\):
  • \(u_1 = v_1\); \(\quad e_1 = u_1/\|u_1\|\)
  • \(u_2 = v_2 - \dfrac{\langle v_2,u_1\rangle}{\langle u_1,u_1\rangle}u_1\); \(\quad e_2 = u_2/\|u_2\|\)
  • \(u_k = v_k - \displaystyle\sum_{j=1}^{k-1}\frac{\langle v_k,u_j\rangle}{\langle u_j,u_j\rangle}u_j\)
Module 7: Eigenvalues and Cayley-Hamilton
Eigenvalues and Eigenvectors
  • Characteristic equation: \(\det(A-\lambda I) = 0\)
  • Eigenvector: \((A-\lambda_i I)\mathbf{x} = \mathbf{0}\)
  • Properties: \(\text{tr}(A)=\sum\lambda_i\); \(\det(A)=\prod\lambda_i\)
  • Cayley-Hamilton: Every matrix satisfies its own characteristic equation. To find \(A^{-1}\) or powers of \(A\), use \(p(\lambda)=0 \Rightarrow p(A)=0\)
✏ Solved — Eigenvalues and Cayley-Hamilton
For \(A=\begin{pmatrix}2&1\\1&2\end{pmatrix}\), find eigenvalues, eigenvectors, and verify Cayley-Hamilton.
1
Char eqn: \(\det(A-\lambda I)=(2-\lambda)^2-1=\lambda^2-4\lambda+3=(\lambda-1)(\lambda-3)=0\) → \(\lambda=1,3\)
2
For \(\lambda=1\): \((A-I)\mathbf{x}=0\) → \(\begin{pmatrix}1&1\\1&1\end{pmatrix}\mathbf{x}=0\) → \(\mathbf{x}=\begin{pmatrix}1\\-1\end{pmatrix}\)
3
For \(\lambda=3\): \(\mathbf{x}=\begin{pmatrix}1\\1\end{pmatrix}\)
4
Cayley-Hamilton: \(A^2-4A+3I = \begin{pmatrix}5&4\\4&5\end{pmatrix}-\begin{pmatrix}8&4\\4&8\end{pmatrix}+\begin{pmatrix}3&0\\0&3\end{pmatrix}=\begin{pmatrix}0&0\\0&0\end{pmatrix}\) ✓
\(\lambda_1=1, \mathbf{x}_1=(1,-1)^T\); \(\lambda_2=3, \mathbf{x}_2=(1,1)^T\)
Gaussian Elimination and Gauss-Jordan
  • Gaussian elimination: Reduce augmented matrix \([A|b]\) to upper triangular form using row operations, then back-substitute
  • Gauss-Jordan: Continue until reduced row echelon form (identity matrix on left); solution reads directly
  • Consistency check: System has solution iff \(\text{rank}(A)=\text{rank}([A|b])\)

Module 1: Descriptive Statistics
Measures of Central Tendency and Dispersion
  • Mean: \(\bar{x} = \dfrac{1}{n}\displaystyle\sum x_i\); Median: middle value; Mode: most frequent
  • Variance: \(\sigma^2 = \dfrac{1}{n}\displaystyle\sum(x_i-\bar{x})^2\)
  • Standard deviation: \(\sigma = \sqrt{\sigma^2}\)
  • Skewness (Pearson): \(\gamma_1 = \dfrac{3(\text{Mean}-\text{Median})}{\sigma}\)
  • Kurtosis: \(\gamma_2 = \mu_4/\sigma^4\); Normal distribution has \(\gamma_2=3\)
Module 2: Random Variables
Key Formulas
  • Discrete: \(\displaystyle E(X)=\sum x\,P(X=x)\), \(\quad \text{Var}(X)=E(X^2)-[E(X)]^2\)
  • Continuous: \(\displaystyle E(X)=\int_{-\infty}^\infty x\,f(x)\,dx\)
  • Covariance: \(\text{Cov}(X,Y)=E(XY)-E(X)E(Y)\)
  • MGF: \(M_X(t)=E(e^{tX})\); \(E(X^n) = M_X^{(n)}(0)\)
  • Joint distribution: \(f_{XY}(x,y)\); marginal \(f_X(x)=\displaystyle\int f_{XY}(x,y)\,dy\)
Module 3: Correlation and Regression
Correlation Coefficient and Regression Lines
  • Pearson's r: \(\displaystyle r = \frac{\sum(x_i-\bar{x})(y_i-\bar{y})}{\sqrt{\sum(x_i-\bar{x})^2\sum(y_i-\bar{y})^2}}\); range: \(-1\leq r\leq 1\)
  • Regression of Y on X: \(y-\bar{y} = b_{yx}(x-\bar{x})\), where \(b_{yx}=r\dfrac{\sigma_y}{\sigma_x}\)
  • Regression of X on Y: \(x-\bar{x} = b_{xy}(y-\bar{y})\), where \(b_{xy}=r\dfrac{\sigma_x}{\sigma_y}\)
  • Relationship: \(r^2 = b_{yx}\cdot b_{xy}\)
  • Rank correlation (Spearman): \(\rho = 1-\dfrac{6\sum d_i^2}{n(n^2-1)}\)

Modules 5–6: Hypothesis Testing
General Procedure
  • Step 1: State \(H_0\) (null) and \(H_1\) (alternative)
  • Step 2: Choose significance level \(\alpha\) (usually 0.05 or 0.01)
  • Step 3: Calculate test statistic
  • Step 4: Find critical value from table
  • Step 5: Decision: reject \(H_0\) if test statistic falls in critical region

Module 6: ANOVA and Design of Experiments
One-Way ANOVA
  • \(\text{SST} = \text{SSB} + \text{SSW}\) (Total = Between groups + Within groups)
  • \(\text{MSB} = \text{SSB}/(k-1)\); \(\text{MSW} = \text{SSW}/(N-k)\)
  • F-statistic: \(F = \text{MSB}/\text{MSW}\)
  • Reject \(H_0\) (all means equal) if \(F > F_{\alpha,k-1,N-k}\)
CRD vs RBD vs LSD
  • CRD (Completely Randomised): Treatments assigned randomly, one-way ANOVA
  • RBD (Randomised Block): Controls one extraneous variable (blocks), two-way ANOVA
  • LSD (Latin Square): Controls two extraneous variables simultaneously, three-way ANOVA
Module 7: Reliability Engineering
Reliability Functions
  • Reliability: \(R(t) = P(T > t) = 1-F(t) = e^{-\lambda t}\) (exponential)
  • Hazard function: \(h(t) = f(t)/R(t) = \lambda\) (constant for exponential)
  • MTTF: \(= \displaystyle\int_0^\infty R(t)\,dt = 1/\lambda\)
  • Series system: \(R_s = R_1\cdot R_2\cdots R_n\)
  • Parallel system: \(R_s = 1-(1-R_1)(1-R_2)\cdots(1-R_n)\)
